# Weather Modeling Simulation Using Parallel Computing

## Overview

This project demonstrates a weather modeling simulation using Python and parallel computing techniques. The simulation models temperature changes across a grid and compares serial and parallel execution methods to evaluate performance improvements.

The project highlights the practical application of High-Performance Computing (HPC) concepts in scientific simulations.

## Problem Statement

Weather simulations often require large-scale computations across geographic grids. Sequential processing can become inefficient for high-resolution grids.

This project addresses the challenge by implementing parallel processing techniques to update multiple grid regions simultaneously and reduce execution time.

## Technologies Used

* Python

* Multiprocessing

## Key Features

* Grid-based temperature simulation
* Serial execution implementation
* Parallel execution using multiprocessing
* Execution time comparison
* Performance analysis
* Temperature heatmap visualization

## Methodology

1. Generate a 2D temperature grid.
2. Update grid cells using neighboring temperature values.
3. Execute simulation using serial computation.
4. Execute simulation using parallel computation.
5. Compare execution times.
6. Visualize temperature distribution using heatmaps.

The parallel implementation demonstrated improved computational efficiency by processing multiple grid sections simultaneously, reducing overall execution time compared to serial execution.
